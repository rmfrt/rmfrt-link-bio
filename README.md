# rmfrt-site

Minimal Astro/Tailwind version of the personal site, currently targeting
`https://rmfrt.xyz` for testing before replacing `rmfrt.com`.

## Commands

```sh
npm run dev
npm run check
npm run build
npm run preview
```

## Environment

```txt
PUBLIC_SITE_URL=https://rmfrt.xyz
PUBLIC_CONTACT_EMAIL=
```

`PUBLIC_CONTACT_EMAIL` is optional. When omitted, no email link is rendered.
